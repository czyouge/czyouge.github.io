我看好或投资的一些Cardano DEX和NFT项目

· 2021-09-24 · # 投资 # Cardano # NFT

Cardano 区块链已经具备运行智能合约的能力，一些项目正在进行编写代码、测试、审计等工作。本文将介绍一些我个人有所了解或较为看好或已经投资的 DEX 和 NFT。

分布式交易所（DEX）

说到区块链和智能合约，DEX 是绕不开的核心话题。一般一条公链都会有一个核心 DEX，比如以太坊有 Uniswap、币安智能链有 PancakeSwap……

目前，Cardano 还没有任何 DEX 上线，但有多个开发团队都以打造出 Cardano 的核心 DEX 为目标。下面简单谈谈我稍微了解的四个 DEX 项目：

  • SundaeSwap：我感觉这是目前 Cardano 生态中最受瞩目的 DEX 项目，但我也认为 SundaeSwap 这个名字是个败笔，容易让人联想到币安链的 PancakeSwap。其创新之处是将采用所谓的 ISO 模式来发行代币，即他们会运行一些质押池（pool），质押到这些池的 ADA 质押者将不会获得 ADA 奖励，而是会收到 Sundae 代币——也就相当于用质押收益购买代币。
  • ErgoSwap：这个 DEX 实际上来自另一条链 Ergo 的开发团队，而 Ergo 则是一条专注于金融领域的公链，另外该链使用的是工作量证明（POW）机制。在我个人看来，Ergo 并没有什么独特之处，很难想象它会有什么竞争力。不过貌似 Ergo 基金会和 Cardano 的主要开发团队（IOG 和 Emurgo）关系不错，比如 Hoskinson 多次称赞 Ergo 开发者的技术实力，所以在 Cardano 社区备受关注——Emurgo 开发的 Yoroi 钱包也支持 Cardano 和 Ergo 两条链。我个人尚未购买 Ergo 的原生代币，但 Ergo 团队的相当大一部分开发重点其实是在 Cardano 生态。除了 DEX，Ergo 还有做 Oracle 和稳定币的计划，所以 Ergo 代币或许有很大的增值潜力。
  • Adax：这个代币在发行时我凑热闹买了一点，目前已经卖掉部分收回成本，但也还留有一些。有未经证实的消息称，这个项目与目前支持其交易的中心化交易所 ExMarkets 之间似乎有某种灰色的关联，其币价也被操控，存在严重注水可能性。争议之下，该项目似乎仍不乏拥趸。我个人继续持有一些观望，反正零成本；看看其团队究竟能不能做出一个交易所。
  • YaySwap：这个我也是凑热闹买了一点，当时好像是第一个 ICO 的 Cardano DEX。目前该团队已经开发了一个功能简陋的到以太坊的桥。其它的炒作信息不多，价格更是多次跌破发行价——不知道是因为力有不逮，还是正专心于技术开发而未注重营销。同样，我持有一点观望。

除了这四个 DEX 项目，另外还有 Cardax、OccamFi、Minswap 等一些 DEX 项目。我没啥了解，就不多谈了。

非同质化代币（NFT）

Cardano 生态内的 NFT 项目（简称 CNFT）正呈井喷式发展，也因此自然是良莠不齐。整体上看，NFT 的主要使用范围还是在于价格炒作，甚至有的项目存在洗钱的嫌疑，而左手倒右手交易（wash trading）地操控市场更是屡见不鲜。未来，NFT 领域不可避免地会出现大浪淘沙般地清洗。所以投资 NFT 必须要格外慎重，一定要做好风险控制，只投入亏掉也不心痛的钱即可，切勿贪多；尤其是投资那些价格较高的 NFT 时。下面简单说说我拥有的一些 CNFT：

  • SpaceBudz：这是 Cardano 链上第一个成功发售的系列 NFT，也因此具有重要的生态文化意义。谈到 Cardano NFT，就不可能避开 SpaceBudz；就像谈论以太坊 NFT 无法避开 CryptoPunk 以及讨论加密货币无法避开比特币一样。
  • Bitlands：一些岛？——其实我也没了解，但是买了两个。
  • CardanoSpace：我个人认为这是目前最被低估的 NFT 项目，因为该项目的每个 NFT 不仅是一个收藏品（collectable），而且还具有使用价值（utility）。持有者可通过交易元数据对每个「坐标」上呈现的内容进行更新。在这个意义上，该项目与现在互联网领域热议的所谓「元宇宙」概念非常契合，未来有着许多让人着迷的可能性。目前该项目还处于相当早期的阶段，很多功能尚未实现，而且其背后的团队（目前好像就两个人）最后究竟能走到哪一步也依然存疑。我已经将自己的博客链接藏在了这个 100x100 的二维方形展览馆中，有兴趣的话可以找找看。
  • Cardano Keys：这个 NFT 项目来自质押池 CKEYS 的运营者。据介绍，这些虚拟钥匙的持有者未来可以得到来自该质押池的收益空投、未来可能会有的交易市场的收益空投并且可以开某些数字盲盒。目前我不清楚收益如何，也完全不知道这些盲盒究竟会是什么，但先投机了几把钥匙。究竟会怎样，等着看看情况咯。
  • Alien Baby Club：挺可爱的。

一些有用的工具

最后介绍一些投资 Cardano 生态项目时可能值得一用的工具：

  • Adapools.org：功能很多的 Cardano 信息工具，能查询到很多 Cardano 链上信息，最主要的是汇集的质押池信息，很有参考价值，比如你可以保存一个搜索结果的书签，这样无需打开钱包也能查看你的质押奖励。另外你也能在其功能菜单中找到许多有意思的数据，比如 Rich List。
  • Cardano Assets 浏览器：方便了解各种链上资产的情况。
  • 两个已有的 CNFT 交易平台：CNFT.io 和 Tokhun.io。
  • CNFT 日历：汇集了一些 CNFT 项目的发售时间等信息。
  • pool.pm：目前 Cardano 的钱包在查看资产方面都表现不佳，这是个不错的替代工具。
  • CNFT Tools：对一些 CNFT 项目进行了分析，能够根据稀有度和价格等条件对 NFT 进行排序。
  • CNFT Analytics：统计了 CNFT 的交易数据，比如一个系列 NFT 的最高、最低和平均单价，有助于了解 CNFT 的买卖情况，辅助持有者合理定价和购买者评估是否划算。
  • @ADAlchemist：一个 Twitter 账户，常常会发布一些对 Cardano 的数据分析结果，可以参考下。
  • @adafloorbot：一个 Twitter Bot，定期发布各 CNFT 项目的最低单价。
