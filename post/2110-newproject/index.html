一个有趣的新项目

· 2021-10-26 · # Cardano # NFT # 日常 # 小说

又来了，恼火！我的脑瓜里似乎偶尔就会出现一个自己觉得很是不错的想法，但最后实践起来却多半并不如意，甚至常常半途而废。但即便知道这一点，我也无法阻止这些新想法的萌发以及自觉良好的感受。现在，我又有了一个新计划。（其实也不算新，我已经隐约有这个想法几个月了。）

说起来很简单，这个计划就是：使用我收集的 NFT 创作一部小说。

毋须讳言，现在我已经拥有了相当数量的 NFT，其中大部分都是 Cardano NFT，另有少量 Ethereum NFT——但由于交易 Ethereum NFT 的成本过高，现在我已经不再关注这个区块链平台上的 NFT。

粗略想来，使用 NFT 角色创作小说还是有些好处的：

  1. 有现成的设定可用。许多 NFT 项目的角色设计特点非常鲜明，用来做人物角色非常可行。
  2. 自己拥有版权的插画。我可以将这些 NFT 图像插入到小说中当插画，再也不用担心版权问题。
  3. 社区效应。一些 NFT 项目拥有非常热情的社区，小说更容易获得读者并得到推广。（遗憾的是，我目前只能使用中文创作，而 NFT 社区主要使用英语。）
  4. 如果可以，作者甚至可以把小说本身也做成 NFT 来出售。（但我的这个项目不会这样做。）

我计划的是写一部不长的科幻小说，我已经做好了简单的大纲，大概会用到多个项目的 NFT，其中至少包括 Spacebudz 和 Baby Alien Club——可以看出故事将发生在宇宙空间之中；我也在思考纳入其它 NFT 项目的可能性，比如作为环境背景或在故事中的游戏中出现。

目前还很难说我会得到什么结果，说不定这又会是一个半途而终的计划。但说这样的话多半只是为我之后的毫无作为而提前铺个台阶，实际并不太大意义，毕竟也实在无人在意。

总之，话已撂下，开始工作吧。
