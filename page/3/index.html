英语词 context 的翻译问题

最近做翻译时又遭遇 context 这个词。这个词的意思很简单直观，但翻译时究竟该选用哪个对应汉语词却需要一些考量。

context，剑桥词典的定义是：

the situation within which something exists or happens, and that can help explain it

即事物存在或发生的情境或环境，并且其能对该事物提供解释。
